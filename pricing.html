FAQ

Common questions.

How does one license work for both desktop and web?

When you buy Pro, you get a single license key. Enter it in the desktop app during installation, and enter the same key in the web app to unlock full access. One purchase, one key, both products.

Does the web app send my financial data anywhere?

Not by default. The web app runs Python locally in your browser using Pyodide — there's no server receiving your numbers. The only automatic network call is license key validation. Pro users can optionally enable cloud sync to back up and move plans across devices — but the plan is encrypted in your browser using your license key before it's uploaded. We receive ciphertext we cannot read. If you never use cloud sync, no financial data ever leaves your device. Full technical details →

What is Founding Member pricing?

$149 is locked in for the first 200 users. Once we hit 200 Founding Members, the price will increase. If you're reading this, it's still available. Founding Members get all future updates free, forever.

Is there a trial before buying?

Yes — the free web app trial lets you use all the modules with one scenario, no account required. That's the best way to see if the depth and interface fit how you think about planning. When you're ready to save scenarios, export, and use the desktop, buy Pro.

What platforms are supported?

The web app works in any modern browser on any device — Mac, iPad, Windows, Linux, Android. The desktop app is available for Windows 10/11 and Linux (AppImage). macOS desktop is on the roadmap.

What if tax laws change after I buy?

That's exactly what Fatboy is built for. You can edit federal and state tax parameters yourself — no waiting for a vendor update. When OBBBA passed in 2025, Fatboy users were updated in minutes. All updates are free for Pro users.

Can I share my plan with my CPA or spouse?

Yes. Pro includes PDF and Excel export, so you can generate professional-quality reports to review with a spouse, share with a CPA or tax advisor, or document your strategy for annual review.

What happens after I buy?

Checkout runs through Stripe. Then you get a receipt, your license key by email, download links for the desktop app, and immediate access to unlock Pro features in the web app.

What payment methods do you accept?

Credit cards and PayPal through Stripe. Secure checkout, nothing stored on our end.

What if I buy it and it is not a fit?

Email us within 30 days and we will make it right. If Fatboy is not the right tool for how you plan, we do not want you stuck with it.
